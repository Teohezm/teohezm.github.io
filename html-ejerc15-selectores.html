Tipos de selectores que entran en examen

1.Clase (class=””)

La etiqueta clase sirve para ,añadiéndola en el "style", puedes crear una clase que puedes añadir a cualquier otra etiqueta del documento y esta cambia su estilo. Por ejemplo el título de este apartado tiene una clase que le pone color y fondo.

2.Identificador (id=””)

La etiqueta clase sirve para ,añadiéndola en el "style", puedes crear una clase que puedes añadir a cualquier una del documento y esta cambia su estilo. Por ejemplo el título de este apartado tiene una clase que le pone color y fondo.

3.Todos los elementos (*)

Permite añadir a todos los elementos del documento un estilo. Por ejemplo en este documento todos los elementos tiene color granate y están alineados al centro.

4.Varios elementos ( p, em, h1, h2)

Permite añadir a algunas etiquetas específicas un estilo. Por ejemplo en este documento los elementos "h1" tiene color verde y fondo rojo.

5.Elementos contenidos en otros elementos (ol li)

Permite añadir a algunas etiquetas de listas un estilo. Por ejemplo en este documento los elementos de las lista tiene colores azul y rosa.

6.Sobrevuleo (p:hover)

Todos los elementos "p" del documento se ponen de color amarillo al pasar el cursor por encima.
